# 🛡️ SecurePass — Premium Cryptographic Suite

SecurePass is a state-of-the-art password management and generation suite designed with a futuristic aesthetic and cryptographic precision. It's not just a tool; it's a visual experience powered by a custom **Electrified Shield Lattice Engine**.

---

## ✨ Key Features

### 🔍 Real-time Entropy Analyzer

- **Instant Feedback**: Watch the shield strength fluctuate as you type.
- **Advanced Metrics**: Evaluates length, character variety, and checks against common "terrible" password databases.
- **Crack Time Estimation**: Get a realistic estimate of how long it would take a brute-force attack to breach your password.

### 🎲 High-Entropy Generator

- **CSPRNG Powered**: Uses `window.crypto.getRandomValues()` for true cryptographic randomness.
- **Customizable Complexity**: Adjust length (up to 64 characters) and toggle character types (Uppercase, Lowercase, Numbers, Symbols).
- **Secure Shuffle**: Implements the Fisher-Yates shuffle algorithm to ensure zero patterns.

### ⚡ Electrified Shield Lattice (Visual Core)

- **Interactive Background**: A high-performance Three.js icosahedron lattice that responds to your input strength.
- **Vibrant Shaders**: Custom GLSL shaders creating cyan, purple, and green gradients with electrical flicker effects.
- **Mouse Interaction**: The shield warps and emits "lightning strikes" toward your cursor based on current password security.

### 📜 Session History & Clipboard

- **Encrypted Local History**: Keeps track of your generated passwords during the session.
- **One-Click Copy**: Integrated clipboard management with smooth UI feedback.
- **Session Clear**: Easily wipe your history when you're done.

---

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla CSS3
- **Logic**: ES6+ JavaScript (Modularized)
- **Engine**: [Three.js](https://threejs.org/) (WebGL)
- **Shaders**: Custom GLSL (Vertex & Fragment)
- **Typography**: Outfit & JetBrains Mono (via Google Fonts)
